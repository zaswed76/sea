

Пользовательские стили
=======================


Добавляем стиль
~~~~~~~~~~~~~~~

#. В каталоге  **'./resource'** в директориях [icons, textures] создать каталог
   с именем стиля

#. В каждом из этих каталогов создать соответствующие иконки и текстуры
   с именами соотвествующими именам из стиля **'grey'**
   **(./resource/icons/grey, ./resource/textures/grey)**

#. В каждый из каталагов - **'./resource/icons/имя_стиля'**, добавить иконку,
   ассоциированую с добавляемым стилем

#. В конфиг - **'./etc/config.json'**, в список с ключом - **'actions_names'**
   добавить базовое имя иконки с расширением

#. Создать файл - **'./css/имя_стиля'**
