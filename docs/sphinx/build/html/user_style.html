Пользовательские стили¶

Добавляем стиль¶

  1. В каталоге ‘./resource’ в директориях [icons, textures] создать каталог с именем стиля
  2. В каждом из этих каталогов создать соответствующие иконки и текстуры с именами соотвествующими именам из стиля ‘grey’ (./resource/icons/grey, ./resource/textures/grey)
  3. В каждый из каталагов - ‘./resource/icons/имя_стиля’, добавить иконку, ассоциированую с добавляемым стилем
  4. В конфиг - ‘./etc/config.json’, в список с ключом - ‘actions_names’ добавить базовое имя иконки с расширением
  5. Создать файл - ‘./css/имя_стиля’
